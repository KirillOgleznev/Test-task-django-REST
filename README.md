# REST backend с использование DRF (Django Rest Framework)

>Новость имеет структуру:
> - Имя;
> - Краткое описание;
> - Полное описание;
> - Тип новости.
> 
>Тип новостей имеет структуру:
>  - Имя;
>  - Цвет.
>

>  {
